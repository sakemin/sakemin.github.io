Inner Peace

2017

Web Media / Sound Design

HTML 5, CSS 3, JavaScript, P5.js, Ableton Live


Try it here : innerpeace.sakem.in


  Inner Peace is a web-based meditation helper. The infinite loop will guide you to the calm state of your mind. It will continuously change it's color along to the background ambient music.

  When you start this program, the infinite loop says hello to you. You can click once to start mindfulness meditation and it will help you to have a deep breathing. Then you can choose the feeling state, and it will help you to deal with the feeling you chose. If you don't start the mindfulness meditation, the infinite loop will keep saying some words that will console you.

  The mindfulness meditation methods are from 'Search Inside Yourself Leadership Institue Korea'.
